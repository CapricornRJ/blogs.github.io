iOS开发之lottie动画

Lottie 是一个可应用于Andriod和iOS的动画库，可以直接使用json实现各种复杂的动画，有了这个第三方框架再也不必为了各种动画的实现而苦恼...

优点：
1、开发成本低，设计同学导出json后可直接使用
2、跨平台，iOS与安卓可使用同一套json
缺点：
1、AE属性支持不够完全
2、系统版本显示：iOS8 及以上，安卓 API14 及以上

一、先看一下官方实现的一部分效果

二、Lottie支持的平台：

支持：iOS、macOS、Android、React Native
当然swift也支持！！！

注意：iOS 支持的版本是 iOS8 及以上，安卓需要 API14 及以上。

三、通过cocoapods集成

1、在podfile中添加：

pod 'lottie-ios'
遇到的问题：
发现UI给的json无法使用，但安卓没有问题，最后进过排查发现pod导入的lottie库代码不是最新的，最后通过指定库的原代码地址导入最新代码
pod 'lottie-ios', :git => 'https://github.com/airbnb/lottie-ios'

2、导入：

pod install

四、相关使用

使用lottie加载json

LOTAnimationView *animation = [LOTAnimationView animationNamed:@"Lottie"];
[self.view addSubview:animation];
[animation playWithCompletion:^(BOOL animationFinished) {
    // Do Something
}];

循环动画

animationView.loopAnimation = true
